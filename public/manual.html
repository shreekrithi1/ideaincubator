üîÑ Idea Lifecycle Workflow

Stage 1: Submission

Employee

  • Employee submits an idea with title, description, category, and t-shirt size
  • Idea enters "Pending Moderation" status
  • Other employees can view and vote on the idea

Stage 2: Moderation

Moderator

  • Moderator reviews idea in Control Tower
  • Can approve (moves to Executive Review) or reject (returns to draft)
  • Provides feedback and comments

Stage 3: Executive Review

Executive

  • Executive views idea in The Boardroom
  • Analyzes SWOT (Strengths, Weaknesses, Opportunities, Threats)
  • Reviews business value score and community votes
  • Authorizes budget with digital signature

Stage 4: Development

System

  • Automatic JIRA story creation
  • Idea status changes to "In Development"
  • JIRA ticket ID linked to idea
  • Development team can begin implementation

üîó JIRA Integration Setup

The system automatically creates JIRA stories when executives approve ideas. Follow these steps to configure the integration:

Prerequisites

  • JIRA Cloud account
  • Project with "Story" issue type enabled
  • API token from Atlassian
  • Admin role in Idea Incubator

Configuration Steps

Generate API Token

Visit Atlassian Account Settings and create a new API token. Copy it immediately as it won't be shown again.

Access Integration Settings

Log in as Admin ‚Üí Navigate to Mission Control ‚Üí Click "Settings" tab

Enter JIRA Configuration

Fill in the following fields:

  • JIRA Instance URL: https://yourcompany.atlassian.net
  • Project Key: Your project key (e.g., INNO, IDEA)
  • Email Address: Your JIRA account email
  • API Token: Paste the token from step 1

Save and Test

Click "Save Configuration" and approve a test idea to verify the integration works correctly.

What Gets Created in JIRA

When an executive authorizes a budget, the system creates a JIRA story with:

  • Summary: Idea title
  • Description: Full idea description with metadata
  • Submitter: Original idea author
  • Category: Product, Process, or Culture
  • Business Value Score: ROI assessment
  • T-Shirt Size: Effort estimate (S, M, L, XL)
  • Labels: innovation, idea-incubator, category tag
üí° Demo Mode: If JIRA is not configured, the system creates mock ticket IDs (DEMO-XXXX) so you can still test the workflow.

Troubleshooting

Issue Solution
SYNC-FAILED ticket ID Check JIRA configuration in Admin settings. Verify API token is valid and project key exists.
401 Unauthorized API token expired or incorrect. Generate a new token and update settings.
400 Bad Request Invalid project key or issue type not available. Verify project settings in JIRA.
DEMO-XXXX ticket ID JIRA not configured. This is expected behavior for testing without JIRA integration.

‚ùì Frequently Asked Questions

General Questions

Q: Who can submit ideas?

A: All employees with an account can submit ideas. There's no limit to the number of ideas you can submit.

Q: How do I vote on an idea?

A: Click the vote button (thumbs up icon) next to any idea in the Innovation Pipeline. You can vote once per idea.

Q: Can I edit my idea after submission?

A: Yes, click the edit icon in the Actions column. However, once an idea moves past moderation, editing may be restricted.

Q: What happens if my idea is rejected?

A: Rejected ideas return to draft status. You'll receive feedback from the moderator and can revise and resubmit.

Technical Questions

Q: Is JIRA integration required?

A: No, the system works in demo mode without JIRA. However, for production use, JIRA integration enables automatic story creation.

Q: How secure is the digital signature process?

A: Digital signatures require typing "APPROVE" exactly and are logged with timestamps and user IDs for audit purposes.

Q: Can I export idea data?

A: Yes, admins can export data as CSV from the Mission Control dashboard.

Q: What browsers are supported?

A: Modern browsers including Chrome, Firefox, Safari, and Edge. Mobile browsers are also supported.

Role-Specific Questions

Q: As a moderator, how do I prioritize which ideas to review first?

A: Ideas are sorted by submission date and vote count. Focus on high-vote ideas and time-sensitive submissions first.

Q: As an executive, can I reject an idea after SWOT analysis?

A: Yes, click the "Reject" button instead of "Authorize Budget". The idea returns to draft status.

Q: As an admin, how often should I check the analytics?

A: Real-time analytics update continuously. Weekly reviews are recommended to track trends and conversion rates.
